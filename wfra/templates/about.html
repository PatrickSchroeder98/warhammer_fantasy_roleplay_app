About the Application

This application allows users to manage their characters from the Warhammer Fantasy Roleplay game. Registration is required to access all website functionalities. Character creation is done by completing a form.

Please note that all information in the form is required. If some fields are not relevant, an acceptable answer should be 'None', '-' or '0' in the case of numerical values. If a character does not have any advanced skills, these fields should be filled with placeholders.

Character details can be accessed from the main list of user characters. In the detail view, users have the option to update or delete a character.

Important: Deleting a character is permanent!

Full software documentation of this application can be viewed on this website .
